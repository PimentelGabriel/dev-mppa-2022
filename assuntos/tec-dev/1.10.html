Sistema de gerenciamento de conteúdo web ZOPE/Plone: principais características, componentes (portlet) e arquitetura Zope (templates, schemas, classes), instalação e configuração de sítios web.

ZOPE: Zope is a free and open source web application server written in the object-oriented programming language Python. Since its release in 1998, Zope continues to grow into many distinct applications, frameworks, libraries and tools. The World of Zope highlights the most important components.
